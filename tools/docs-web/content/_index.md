## Quick Start

Get started with Manuscript in minutes:

```bash
# Clone and build
git clone https://github.com/manuscript-lang/manuscript.git
cd manuscript
make build-msc

# Create your first program
echo 'print("Hello, Manuscript!")' > hello.ms

# Run it
./build/msc hello.ms
```
